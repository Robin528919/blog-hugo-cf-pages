# 微信公众号内容审核规则

## 敏感词分类

### 一级敏感词（必定触发 errcode 45166）

**海外通讯平台名称：**
- Telegram、WhatsApp、Discord、Signal、Line
- 适配策略：根据上下文改写
  - 列举平台时 → "多个海外主流即时通讯平台"
  - 对比场景时 → "其他即时通讯工具"
  - 作为功能说明时 → 直接省略具体名称，用功能描述替代

**敏感技术术语：**
- VPN、翻墙、科学上网、梯子、代理（指网络代理）
- 逆向工程、逆向协议、破解、越狱、Root
- 适配策略：
  - "逆向协议" → "非官方对接方案" / "第三方技术方案"
  - "VPN" → 视上下文删除或改为"网络工具"

**账号安全相关：**
- 封号、封禁、被封、账号风险
- 适配策略：
  - "封号风险" → "稳定性问题" / "合规风险"
  - "被封" → "受到限制"

### 二级敏感词（可能触发，视上下文）

**竞品品牌（大量提及时）：**
- 具体竞品 App 名称密集出现可能触发
- 适配策略：首次提及保留，后续用"该平台"等代称

**政治/时事敏感：**
- 不在本 Skill 范围内，由人工判断

## 微信平台限制

| 限制项 | 值 |
| ------ | ----- |
| 标题长度 | 64 字符（个人订阅号） |
| 摘要长度 | 120 字符 |
| 正文长度 | 约 20,000 字符 |
| 外链 | 自动过滤，仅白名单域名可保留 |
| 图片格式 | 不支持 SVG，需转 PNG/JPG |
| author 字段 | 个人订阅号不可用（errcode 45110） |

## 风格适配建议

### 博客 vs 微信的差异

| 维度 | 博客 | 微信 |
| ------ | ------ | ------ |
| 阅读场景 | 桌面浏览器，专注阅读 | 手机碎片化阅读 |
| 段落长度 | 可较长（5-8 行） | 宜短（2-4 行） |
| 代码块 | 完整展示 | 精简为关键片段 |
| 链接 | 可点击跳转 | 被过滤，需转脚注 |
| 语气 | 可偏技术化 | 适当口语化 |
| 配图 | SVG 矢量图 | PNG（脚本自动转换） |
