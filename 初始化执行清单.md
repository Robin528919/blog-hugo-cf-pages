## 待完成（缺少 Cloudflare API Token）

5. Cloudflare CLI 创建 Pages 项目

执行命令：

```bash
export CLOUDFLARE_API_TOKEN=你的token
wrangler pages project create blog-hugo-cf-pages --production-branch main
```

创建成功后可继续：

```bash
wrangler pages deployment create ./public --project-name blog-hugo-cf-pages
```
